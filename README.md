# 📊 DecodeLabs Data Analytics Internship – Project 1

## 🧹 Data Cleaning & Preparation

### 📌 Project Overview

This project was completed as part of the **DecodeLabs Data Analytics Internship**. The objective was to clean and prepare a raw dataset by handling missing values, removing duplicate records, correcting data formats, and validating data quality using Python and Pandas.

---

## 🎯 Objectives

* Identify missing values
* Handle null values
* Remove duplicate records
* Standardize date formats
* Clean text data
* Validate dataset quality
* Export the cleaned dataset

---
