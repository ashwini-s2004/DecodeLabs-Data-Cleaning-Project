# 📌 Project Overview

The **Data Cleaning & Preparation** project was completed as part of the **DecodeLabs Data Analytics Internship (Batch 2026)**.

The primary goal was to transform a raw, inconsistent dataset into a **clean, structured, and analysis-ready dataset** using Python and Pandas.

This project demonstrates practical skills in data preprocessing, validation, and exploratory analysis.

# 📊 Data Cleaning Process

### ✅ Dataset Loaded

The raw Excel dataset was successfully imported using Pandas.

### ✅ Missing Values

* Identified missing values
* Filled numerical values using Median
* Filled categorical values using suitable replacements

### ✅ Duplicate Records

* Detected duplicate rows
* Removed duplicate entries
* Verified dataset uniqueness

### ✅ Date Formatting

* Converted dates into a consistent format
* Ensured proper datetime representation

### ✅ Text Cleaning

* Removed unwanted spaces
* Standardized text formatting

### ✅ Final Validation

The dataset was validated to ensure:

* No Missing Values
* No Duplicate Records
* Correct Data Types
* Consistent Formatting
